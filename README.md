# 💬 Real-Time Chat Application

A full-stack real-time chat application built using Spring Boot, WebSocket (STOMP), and MongoDB.  
This project demonstrates live messaging between multiple users with persistent chat history.

---

## 🚀 Features

- 🔴 Real-time messaging using WebSocket & STOMP
- 👥 Multi-user chat support (multiple browser tabs)
- 💾 Chat persistence using MongoDB
- 🔄 Load previous messages on refresh
- 🟢 User Join / Leave notifications (in progress)
- 🔐 JWT Authentication (planned)
- 💬 Structured messaging (sender + content)

---

## 🛠️ Tech Stack

- Backend: Spring Boot
- WebSocket: STOMP Protocol
- Database: MongoDB
- Frontend: HTML, CSS, JavaScript
- Build Tool: Maven
